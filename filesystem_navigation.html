To acquire the next web address, you must learn how to navigate filesystems using the terminal.

To create the filesystem to navigate, open a bash repl project by going to the following web address: https://repl.it/languages/bash. You will only be using the terminal window on the right-most third of the screen.

Once you are there, type the following commands. These commands download the compressed file holding the filestructure, uncompress it, and remove the compressed file. Once you have run these commands, you can navigate the directories (aka folders) to look through the files to figure the next web address.

wget https://raw.githubusercontent.com/mjdargen/ctf_FS/main/ctf_FS.tar.gz

tar -xzf ctf_FS.tar.gz

rm -rf ctf_FS.tar.gz
