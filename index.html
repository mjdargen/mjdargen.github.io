writersblock.exe
				
		

	

	

		

		Have writer's block? Machine learning can help! Try using this Google Colaboratory Notebook I created that shows how to use OpenAI’s GPT-2, a state-of-the-art machine learning model for text generation. Colab allows you to use a virtual machine with a GPU to run machine learning models much more efficiently without burdening your hardware. You can run this model in Colab from any computer. Just follow the step by step instructions described in the project to generate your own text!
		


		Below is an example of text generated using GPT-2 with the prompt: "Harry turned to Frodo and gave him a look."
		


		

			Harry turned to Frodo and gave him a look. "You've done all you could. I don't see why we should be worried about your family."

			"I don't think I need to explain that," Frodo said, "I've told you before, I'm not really your brother."

			"It's not a secret, really," Sam said, "I've been telling you that since I was a little boy, and I think you know why."

			"I know that's not true," Frodo said, "but it's not a secret that I am a Hobbit."

			"You are a Hobbit," Sam said.

			"I am a Hobbit," Frodo repeated.

			"And I am a man," Sam said.

			"And I am a wizard," Frodo said.
